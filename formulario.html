la etiqueta placeholder="Nombre deseado" se usa para mostrar el nombre deseao dentro del recuadro

Una vez el recuadro muestre el nombre con el placeholder se puede autocompletar el relleno. Si deseo que esto no suceda se debe agregar la siguiente etiqueta autocomplete="off"

Etiqueta fieldset:  

Esta etiqueta se usa para encerrar el formulario dentro de un recuadro. Si se desea que tenga el recuadro un nombre para distinguirlo se debe usar la etiqueta legend

Formulario de selección de opción

A continuacion se mostrará como se realizan los formularios de seleccion, los cuales son utilizados por ejemplo para seleccionar el sexo de la persona.
Se usa el type="radio" name="" id="" donde dice nombre se agrega el tipo de recuadro al que sera unido el texto y el ID se usa para una vez se clickee el nombre del formulario principal que en este caso se llama Orientacion se vaya directamente a los puntos que se deben rellenar.
Al final del cierre de la etiqueta se pone el nombre como se desee llamar el punto.


se usa la palabra checked para dar la orden que se seleccione donde se coloque. Tambien se puede usar el autofocus para hacer forma de un sombreado en los recuadros que debe seleccionar el usuario.

Lista desplegable

Para formar esta lista desplegable, se debe iniciar con la etiqueta SELECT y luego la etiqueta OPTION que es la encargada de mostrar los nombres que se desean mostrar en la lista desplegable.
Si deseo que un nombre de la lista desplegable se muestre por defecto se debe poner el indicador SELECTED en la palabra que se desea mostrar.
